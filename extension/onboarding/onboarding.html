⚠️ Paso Vital Requerido

Para que FloatCLI pueda integrar una terminal real en tu navegador, requiere enlazarse con tu sistema operativo. Descarga el motor local (Native Host) desde nuestro repositorio oficial para completar la instalación.

Descargar Native Host desde GitHub

Instrucciones de configuración:

  1. Extrae el archivo descargado en tu ordenador (en una ubicación permanente).
  2. Haz doble clic en el archivo install_windows.bat para enlazarlo con Chrome.
  3. ¡Listo! Abre la extensión haciendo clic en su icono en el navegador.
